# 搜索资产{#search-for-assets}

搜索 Adobe Experience Cloud 库中的资产，以便在整个组织或在特定的文件夹中查找资产。

使用搜索栏在 Experience Cloud 库中搜索资产。搜索期间将会查找文件名。
