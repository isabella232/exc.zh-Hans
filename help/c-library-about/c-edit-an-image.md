# 编辑图像{#edit-an-image}

运用内置于 Adobe Experience Cloud 库中的基本图像编辑器，直接在库中即时编辑图像。

要在 Experience Cloud 库中编辑图像，请执行以下操作：

1. 单击要进行编辑的图像。
1. 单击“编辑图像”。保存图像时，该图像将另存为新版本。

   此时将会打开图像编辑器，您可以在此编辑图像。
